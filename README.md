## 프로젝트 개요
해당하는 날짜의 **박스오피스** 순위를 조회하고 영화를 검색하거나 영화에 대한 **자세한 정보(예고편, 스틸샷, 포스터, 배우 정보, 줄거리, 댓글(평점) 서비스)** 제공합니다.
